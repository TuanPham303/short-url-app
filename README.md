## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node server.js` command.
- On the brownser, copy and paste 'localhost:8080' and hit enter to run   the application.
